Detailed Description

This file contains the main() code for a program which runs the ME405 board for ME405 lab 1. This program currently uses the H-bridge chips on the board to set the power of the two motors and let them run for two seconds then brakes them, waits for two seconds, and then runs them again in the opposite direction.
