void task_user::run ( void  )

This method is called by the RTOS once to run the task loop for ever and ever.

This task interacts with the user so that a motor may be selected, the power for that motor may be set, the intensity of braking may also be set, along with a full braking option. It includes two help messages: Initial menu to choose global options, a motor menu to select power or braking intensity.
