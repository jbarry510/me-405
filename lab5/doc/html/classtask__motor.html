void task_motor::run ( void  )

This method is called by the RTOS once to run the task loop for ever and ever.

This method is called once by the RTOS scheduler. Each time around the for (;;) loop, it sets the power of the two motors and lets them run for two seconds then brakes them, waits for two seconds, and then runs them again in the opposite direction.
