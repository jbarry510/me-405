void task_imu::run ( void  )

This method is called by the RTOS once to run the task loop indefinetly.

This method is called once by the RTOS scheduler. Each time around the for (;;) loop, it instatiates a new IMU object, giving it a pointer to the serial port, and requesting to print the system status of the IMU and return the Euler angles for printing.

Creates a new IMU object

Initializes the Euler angle variables

Main task loop

Calls the system status method in the imu_drv which prints a message regarding the status

Gets the Euler angle variables by calling the getEulerAng method in the imu_drv.

Prints the Euler angle variables to the serial port
