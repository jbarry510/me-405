void task_control::run ( void  )

This method is called by the RTOS once to run the task loop indefinetly.

This method is called once by the RTOS scheduler. Each time around the for (;;) loop, it instantiates the PID objects for the motors and sets the velocities of the motors based on the shared setpoint variable. The circular and linear routing calculations are called and used here to set the proper setpoints for the motor and servo.
