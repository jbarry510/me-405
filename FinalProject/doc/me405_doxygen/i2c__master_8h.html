Detailed Description

This file contains a base class for classes that use the I2C (also known as TWI) interface on an AVR. The terms "I2C" (the two means squared) and "TWI" are essentially equivalent; Philips has trademarked the former, and Atmel doesn't pay them a license fee, so Atmel chips that meet exactly the same specification are not allowed to use the "I2C" name, even though everything works the same.
