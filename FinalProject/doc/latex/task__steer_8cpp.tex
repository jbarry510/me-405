\subsection{Detailed Description}
This file contains the code for a task that instatiates the servo object and updates the servo position based on the setpoint. The max setting range for the servo is 2000 to 4000 which corresponds to the 1.\-0ms and 2.\-0 ms pulse width generated by the P\-W\-M.
